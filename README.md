## festigo-app
This project will house our festival app for our complex software system project.

## Running the code
- Run the testharness file which calls our API endpoints. The test data will be printed in the IDE terminal. 

## API Endpoints
### EventAPI Endpoints
  - addEvent
  - updateEvent
  - deleteEvent
  - addToCalendar
  - removeFromCalendar
### FestivalAPI Endpoints
  - postFestival
  - getFestivalById
  - getAllFestivals
  - updateFestival
  - deleteFestival
### LoginAPI Endpoints
  - getUsersById
  - postUsers
  - updateUser
  - deleteUser
### MasterScheduleAPI Endpoints
  - getMasterSchedule
### PersonalScheduleAPI Endpoints
  - getPersonalSchedule
  - addToPersonalSchedule
